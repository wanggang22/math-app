# Mac 卸载软件指南

## 方法一：直接拖拽删除（最常用）

1. 打开 **Finder** → **应用程序**
2. 找到要卸载的应用
3. 将其拖拽到**废纸篓**，或右键 → **移到废纸篓**
4. 清空废纸篓

## 方法二：Launchpad 删除

1. 打开 **Launchpad**（F4 或 Dock 中的火箭图标）
2. 长按应用图标，直到图标抖动
3. 点击左上角的 **"×"** 按钮
4. 确认删除

> 注意：此方法仅适用于 App Store 下载的应用

## 方法三：使用应用自带卸载器

部分软件（如 Office、Adobe 系列）自带卸载程序：

- 打开 **应用程序** 文件夹，找到该软件的文件夹
- 运行其中的 **Uninstaller** 程序

## 方法四：使用第三方卸载工具（推荐彻底清理）

直接拖拽删除可能残留配置文件，推荐使用：

| 工具 | 特点 | 价格 |
|------|------|------|
| **AppCleaner** | 自动找到关联文件 | 免费 |
| **CleanMyMac X** | 功能强大，界面友好 | 付费 |

## 手动清理残留文件

拖拽删除后，可手动清理以下位置的残留文件：

```
~/Library/Application Support/
~/Library/Preferences/
~/Library/Caches/
~/Library/Logs/
```

在 Finder 中按 `Cmd + Shift + G`，粘贴路径进行访问，删除对应应用的文件夹。

## 总结

| 场景 | 推荐方法 |
|------|---------|
| 普通应用 | AppCleaner 卸载 |
| App Store 应用 | Launchpad 删除 |
| 专业软件（Adobe/Office） | 使用自带卸载器 |
